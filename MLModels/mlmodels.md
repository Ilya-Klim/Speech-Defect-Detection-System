# ML-модели для классификации дефектов речи

## Цель эксперимента

Сравнить несколько классических алгоритмов машинного обучения на задаче бинарной классификации произношения скороговорок (хорошо / плохо) по MFCC-признакам.

## Данные

- **Общее количество записей:** 2773
- **Признаки:** 255 MFCC-коэффициентов (40 MFCC + дельты + дельты-дельты + спектральные признаки)
- **Целевая переменная:** 0 – «хорошо», 1 – «плохо»
- **Разделение:** Train – 80% от всех данных, внутри train 80% обучение, 20% валидация. Тестовая выборка – 20% (555 записей).

## Предобработка

- Приведение всех аудио к 16 кГц и длительности 10 секунд (центральная обрезка / дополнение тишиной).
- Стандартизация признаков (`StandardScaler`).

## Модели и гиперпараметры

Для каждой модели проводился поиск гиперпараметров с помощью `RandomizedSearchCV` (5‑fold стратифицированная кросс‑валидация, метрика – `f1` для класса «плохо»).

### Логистическая регрессия (Logistic Regression)

- **C** = 0.00336
- **solver** = liblinear
- **Результаты на тесте:**  
  Accuracy = 0.701, F1 (плохо) = 0.595, ROC‑AUC = 0.754

### Случайный лес (Random Forest)

- **n_estimators** = 200  
- **min_samples_split** = 10  
- **min_samples_leaf** = 4  
- **max_depth** = 5  
- **Результаты на тесте:**  
  Accuracy = 0.692, F1 (плохо) = 0.534, ROC‑AUC = 0.724

### SVM (RBF)

- **C** = 1.0 (базовый)  
- **gamma** = scale  
- **Результаты на тесте:**  
  Accuracy = 0.741, F1 (плохо) = 0.621, ROC‑AUC = 0.797

*Примечание:* Подбор гиперпараметров для SVM дал C = 0.298, gamma = 0.00144, но метрики практически не изменились, поэтому для практического использования оставлена базовая SVM.

### LightGBM

- **num_leaves** = 47  
- **learning_rate** = 0.0322  
- **n_estimators** = 140  
- **min_child_samples** = 43  
- **Результаты на тесте:**  
  Accuracy = 0.728, F1 (плохо) = 0.591, ROC‑AUC = 0.759

### CatBoost

- **depth** = 6  
- **learning_rate** = 0.1  
- **iterations** = 100  
- **l2_leaf_reg** = 5  
- **Результаты на тесте:**  
  Accuracy = 0.744, F1 (плохо) = 0.545, ROC‑AUC = 0.752

## Заключение

- SVM на MFCC‑признаках даёт стабильный результат (F1 = 0.62, ROC‑AUC = 0.80) и будет выбрана в качестве основной модели для веб‑сервиса, если связка спектрограмма + cnn покажет результат хуже.
- Другие модели уступают SVM другим метрикам.
- На основе анализа лучшей модели отобраны 5 скороговорок с высокой точностью предсказания и оптимальным количеством записей:
  1. **рлш** – «Наш голова вашего голову головой переголовил, перевыголовил»  
  2. **стр** – «Хитрую сороку поймать морока, а сорок сорок — сорок морок»  
  3. **трш** – «Шарики шарикоподшипника шарят по подшипнику»  
  4. **стрлш** – «Токарь Раппопорт пропил пропуск, рашпиль и суппорт»  
  5. **чстрл** – «Четыре чёрненьких чумазеньких чертёнка чертили чёрными чернилами чертёж чрезвычайно чисто»
